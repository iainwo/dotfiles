# dotfiles

Scripts and configurations for bootstrapping a new nix environment.

___

To install run:

```bash
curl -s https://raw.githubusercontent.com/iainwo/dotfiles/main/bootstrap.sh | bash
```
___

Programs Installed:
- git
- zsh
- tmux
  - tmux plugin manager
- neovim
